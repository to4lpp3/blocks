# TextGradient

```js
import { TextGradient } from '@blocks/kit'
```

The TextGradient is a tiny little component to set a text gradient. You
can also optionally pass it `random={true}` to get a new gradient each time
it renders.

<TextGradient start='#21D4FD' end='#B721FF'>

# Hello, world!

</TextGradient>

## Table of properties

Prop | Default | Description
---- | ------- | -----------
`start` | `rebeccapurple` | Starting color for the gradient
`end` | None | Ending color for the gradient
`stop` | `60%` | Location where the ending color gradient takes over
`angle` | `90` | Angle of the gradient
`random` | `false` | Random gradient each time.
